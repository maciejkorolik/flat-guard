# Enriched Listing Artifacts

This directory is for local Google-based enrichment outputs produced by:

- `node scripts/enrich-rental-listings-google.mjs ...`

Typical files:

- `olx_wroclaw_rental_enrichments_<timestamp>.jsonl`
- `olx_wroclaw_rental_enrichments_<timestamp>.csv`
- `olx_wroclaw_rental_enrichments_<timestamp>.meta.json`

Checked-in outputs are intentionally avoided so local reruns do not pollute commits.
